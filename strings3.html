Here are a few examples: 1 .length returns the number of characters in a string. Example: "hello".length returns 5. 2 .concat() joins two or more strings together. Example: "hello, ".concat("world!") returns "hello, world!" 3 .toUpperCase() converts a string to uppercase letters. Example: "hello".toUpperCase() returns "HELLO" 4 .toLowerCase() converts a string to lowercase letters. Example: "HELLO".toLowerCase() returns "hello" 5 .indexOf() returns the first index of a specified character or substring in a string. Example: "hello".indexOf("l") returns 2 6 .lastIndexOf() returns the last index of a specified character or substring in a string. Example: "hello".lastIndexOf("l") returns 3 7 .slice() returns a specified portion of a string. Example: "hello".slice(1, 4) returns "ell" 8 .substring() is similar to .slice() but it is not support negative index. "hello".substring(1, 4) returns "ell" 9 .replace() replaces a specified character or substring in a string with another character or substring. Example: "hello".replace("l", "x") returns "hexxo" 10 .split() splits a string into an array of substrings. Example: "hello world".split(" ") returns ["hello", "world"]
